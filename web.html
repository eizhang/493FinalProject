The Impossibruuuuu Quiz

Samantha Schwartz - Emmie Zhang - Ori Lindner - Bryant Vergara

Background

The Impossibruuuu Quiz is a web-based game to challenge users to think outside the box when tackling, on the outside, what seems like a standard point-and-click quiz. Our quiz is designed to have non-trivial answers, testing how the user interacts with the DOM and knowledge of pop-culture and other topics.

Gameplay

The quiz is a compilation of unique levels designed to test how the user interacts with the page. All interactions are based on point-and-click events. The user begins with 3 lives, and with each incorrect click the user will lose a life. Once all of the lives have been used the game starts from the beggining, so the user must be sure to remember the correct answers on the next turn if they want to beat all 16 levels.
